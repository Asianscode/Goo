goo.get_division_angles(axis)¶

This function returns 2 angles associated with the division axis of a cell: The angle between the the division axis and the z-axis (phi, in spherical coordinates) and the angle projected on the xy-plan (theta)

Parameters

axis – the normal (e.g. long axis) to the division plane

expressed as (x, y, z)

Returns

(phi, theta). phi is the angle between the division axis and the z-axis.

theta is the angle projected on the xy plane

goo.select_translate_cell(obj, COM, division_axis)¶

This function is called after dividing the mother cell. We compare the center of mass of the original mother cell to that of the corresponding daughter cell. We calculate the signed distance between the daughter cell’s center of mass and the original plane of division. If the distance is positive, the function returns 0 and this daughter cell will be translated later. If the distance is negative, the function returns 1 and the other daughter cell will be translated later

Parameters

  • obj – the Blender mesh to divide in two

  • COM – center of mass

  • division_axis – division axis of cell (cleavage plane is

perpendicular to division axis)

Returns

0 if the daughter cell is a positive distance away from

the plane of division and 1 otherwise
