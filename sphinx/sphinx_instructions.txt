To make Sphinx documentation
-edit .rst files to change content.  Index.rst is homepage.
-open Goo.py
-comment out import bpy, bmesh, mathutils, numpy as np
-save
-from Terminal in Goo/Sphinx directory, type 'make html'
-check for errors. Sometimes re-running will fix errors
-this should remake the html files in sphinx/_build
-uncomment out import bpy, bmesh, mathutils, numpy as np
-save Goo.py
-copy all the files in /sphinx/_build/html/ to /docs/
-use Git to commit the sphinx pages in docs/ to Github to appear as Github Pages
